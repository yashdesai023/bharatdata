# BharatData Playground

The official web interface for querying and downloading clean Indian public data from the National Crime Records Bureau.

## Overview
This application is designed for journalists, researchers, and citizens who want zero-setup access to public datasets. Under the hood, this dashboard consumes the `@bharatdata/typescript-sdk` and `@bharatdata/api`.

## Local Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Features
*   **Instant Access**: Formatted dropdown capabilities to select exactly the state and year of data you want. No code required.
*   **Attribution Proof**: Contains the metadata to credit the authentic NCRB source immediately.
*   **One-Click Download**: Provides pure client-side structured JSON files and proper Excel-supported UTF-8 CSV downloads.
